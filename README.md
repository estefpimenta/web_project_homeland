# Projeto 5: Tripleten web_project_homeland

A página web que mostra as cidades natais de alguns funcionários da TripleTen. Ela foi criada usando HTML e CSS, com base no figma disponibilizado.

## Recursos do projeto

- HTML5 semântico
- Flexbox
- Posicionamento
- @font-face
- Metodologia BEM
- Responividade
